# 技術面接クイズアプリ

HTML、JavaScript、CSSのみで作成された技術面接対策のクイズアプリケーションです。

## 特徴

- 4つのカテゴリ（Web基礎、Web応用、Webセキュリティ、AWS）
- 各カテゴリ10問の4択問題
- LocalStorageによる進捗保存
- レスポンシブデザイン

## 使い方

1. `index.html` をブラウザで開く
2. カテゴリを選択
3. 問題に回答
4. 結果を確認し、復習機能で理解を深める

## 技術スタック

- HTML5
- CSS3（Flexbox、Grid）
- Vanilla JavaScript（ES6+）
- LocalStorage API
